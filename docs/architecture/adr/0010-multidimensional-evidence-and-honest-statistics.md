# ADR 0010: Multidimensional Evidence and Honest Statistics

- Status: Accepted
- Date: 2026-07-23

## Context

Lesson completion, exposure, recall, listening, reading, and production are not
interchangeable measures.

## Decision

Record evidence per objective, skill dimension, modality, and optional
representation. Keep curriculum progress and review scheduling separate.
Statistics show samples, periods, and whether values are measured,
self-reported, or estimated.

Never infer CEFR mastery from completion, exposure, or a single global score.

## Consequences

Storage and queries are more detailed, but review and learner feedback remain
truthful. Raw evidence supports future algorithm changes.

## Rejected alternatives

- Global fluency percentage.
- Exposure-as-mastery.
- Spoken accuracy without speech evidence.
